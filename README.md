# 👕 Wear the Love : 사랑을 입다
> IoT 기반 의류 케어 스마트 옷장 프로젝트

---

## 📌 프로젝트 개요
**Wear the Love**는 패스트 패션으로 인한 의류 낭비 문제와  
의류 보관 환경에서 발생하는 위생·관리 문제를 해결하기 위해 기획된  
**IoT 기반 의류 관리 서비스**입니다.

Android 앱과 Arduino(ESP8266), Firebase를 연동하여  
의류 보관 환경을 원격으로 제어하고,  
사용자가 보유한 의류 데이터를 기반으로 한 관리 경험을 제공하는 것을 목표로 합니다.

---

## 🎯 기획 배경
- 패스트 패션 문화로 인한 무분별한 의류 생산·소비 증가
- 의류 보관 환경(습도, 세균)에 대한 관리 필요성
- 날씨에 따른 옷 선택의 번거로움
- 세탁 정보 부족으로 인한 의류 수명 단축

기술을 활용해 **의류를 오래, 위생적으로, 효율적으로 관리할 수 있는 방법**을 제안하고자 본 프로젝트를 진행했습니다.

---

## 🛠 주요 기능

### 1️⃣ 원격 의류 보관 환경 제어
- 옷장 내부 **온·습도 실시간 모니터링**
- 사용자 설정에 따른 **가습 / 제습 제어**
- **UV 살균 램프 타이머 설정 및 원격 제어**

### 2️⃣ 의류 데이터 관리
- Firebase Realtime Database 기반 의류 데이터 저장 구조 설계
- 옷장 페이지에서 **의류 목록 조회 화면 구성**
- 의류별 세탁 정보 확인 기능 제공  
  *(현재는 더미 데이터를 활용해 UI 및 데이터 흐름 검증)*

### 3️⃣ 날씨 기반 옷차림 추천 (설계 단계)
- Weather API를 활용한 실시간 날씨 데이터 수집
- Python 기반 옷차림 추천 로직 및 모델링 설계
- 사용자 의류 속성(소재, 색상, 계절감)을 고려한 추천 구조 구성  
  *(앱 연동은 구조 설계 단계까지 진행)*

---

## 🧪 사용 기술

### Software
- **Android** (Java, Android Studio)
- **Firebase Realtime Database**
- **Python** (추천 로직 모델링)
- Weatherbit API

### Hardware
- **Arduino ESP8266**
- 온·습도 센서 (DHT11)
- UV 살균 램프
- 가습기 / 제습기 모듈
- NFC 리더기 (확장 고려)

---

## 👩‍💻 담당 역할
### Android 앱 전반 기능 구현
- 회원가입 및 로그인 기능 설계 및 구현
- Weatherbit REST API를 활용해 사용자 위치 기반 날씨 데이터 수집 및 의류 추천에 활용 가능한 정보로 가공하여 제공
- ESP8266과의 통신을 위해 습도 제어 및 UV 살균 램프 제어 기능 구현
- UV 램프 동작 시간 타이머 및 알림 기능 구현
- Firebase를 활용한 사용자 정보, IoT 상태 데이터 관리 및 실시간 반영

---

## 🚧 한계 및 개선 방향
- NFC 기반 의류 등록 기능은 구조 설계 및 UI 검증 단계까지 진행
→ NFC에 등록된 데이터 또한 태깅시 값을 읽어오지 못 한 상태
- 옷차림 추천 모델은 Python 단에서 설계 완료, 앱 연동 미완
→ Python 로직 또한 추가 보완 필요
- 향후 실제 사용자 데이터 기반 추천 정확도 개선 예정

## 🌱 기대 효과
- 의류 보관 환경 개선을 통한 의류 수명 연장
- 원격 제어를 통한 사용자 편의성 향상
- 불필요한 의류 소비 감소로 지속 가능한 소비 문화 기여
